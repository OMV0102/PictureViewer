Краткая справка по PictureViewer.

Данное приложение предназначено для просмотора изображений, хранящихся на удаленном сервере. Удаленный сервер запускается отдельным приложением PictureViewerServer.

Справа отображается список изображений. хранящихся на сервере. Поддерживаются следующие форматы: jpg, png, gif, jpeg, bmp.

С помощью кнопки "+" пользователь может добавить новые изображения на сервер.

С помощью кнопки "Обновить" пользователь может получить список изображений с сервера заново.

С помощью контекстного меню пользователь может сохранить изображение локально на свой компьютер или удалить изображение с сервере.

Изображение можно редактировать:
* отобразить зеркально по вертикали/горизонтали;
* повернуть по/против часовой стрелки;

Переключившись в режим изменения масштаба, можно зрительно увеличить и уменьшить размер изображение для более детального рассмотрения.
